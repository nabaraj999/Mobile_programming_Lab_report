# Mobile Programming Lab Reports

This repository contains the lab exercises for **Mobile Programming**. Each lab includes its description, purpose, and corresponding files.

---

## LAB 1: Simple Calculator
**Description:** Develop an Android app to make a simple calculator.  
**Files:**
- `activity_lab_one.xml` – Layout for the calculator activity  
- `LabOne.java` – Java code for calculator functionality  

---

## LAB 2: Custom ListView and GridView
**Description:** Create a Custom ListView and GridView with an icon.  
**Files:**
- `activity_lab_two.xml` – Main activity layout  
- `list_item.xml` – Layout for each item in the list/grid  
- `LabTwo.java` – Activity Java file  
- `CoustomAdapter.java` – Adapter for list/grid view  

---

## LAB 3: Recycler View with Icon and Event
**Description:** Create a RecyclerView with icon and event handling.  
**Files:**
- `ItemModel.java` – Data model class  
- `ItemAdapter.java` – RecyclerView adapter  
- `row_item.xml` – Layout for each RecyclerView item  
- `activity_main.xml` – Main activity layout  
- `MainActivity.java` – Main activity Java file  

---

## LAB 4: Context Menu and Popup Menu
**Description:** Demonstrate context menu and popup menu.  
**Files:**
- `LabPopupMenuActivity.java` – Activity for popup menu  
- `activity_lab_popup_menu.xml` – Layout for popup menu activity  
- `LabContextMenuActivity.java` – Activity for context menu  
- `activity_lab_context_menu.xml` – Layout for context menu activity  
- `res/menu/popup_lab_menu.xml` – Menu XML file  

---

## LAB 5: Google Map Application
**Description:** Develop a Google Map application using API and show 2 nearest places.  
**Files:**
- `activity_lab_google_map.xml` – Layout for map activity  
- `LabGoogleMapActivity.java` – Activity Java file  

---

## LAB 6: Fragment Demo
**Description:** Write a program that adds fragments in different ways.  
**Files:**
- **Activity:** `FragmentDemoActivity.java`  
- **Fragment 1:** `FirstFragment.java`  
- **Fragment 2:** `SecondFragment.java`  
- **Activity Layout:** `activity_fragment_demo.xml`  
- **Fragment 1 Layout:** `fragment_first.xml`  
- **Fragment 2 Layout:** `fragment_second.xml`  

---

## LAB 7: API Demo to Display Student Information
**Description:** Develop an Android app to decode and display information using an API which retrieves data from a student table `(id, name, age, faculty)`.  
**Files:**
- `StudentModel.java` – Data model class  
- `StudentAPI.java` – API class  
- `StudentAdapter.java` – RecyclerView adapter  
- `activity_student_api_demo.xml` – Activity layout  
- `row_student_item.xml` – Layout for each student item  
- `StudentApiDemoActivity.java` – Activity Java file  

---

## LAB 8: Custom Dialog Box
**Description:** Create an Android app with a custom dialog box. The dialog asks for student details `(Name, Roll, Result, Grade)` and displays the data in the activity when saved.  
**Files:**
- **Activity:** `StudentDialogActivity.java`  
- **Dialog Layout:** `dialog_student_input.xml`  
- **Activity Layout:** `activity_student_dialog.xml`  

---

## LAB 9: CRUD Operations with SQLite
**Description:** Write a program to perform CRUD operations in Android using SQLite.  
**Files:**
- `StudentDatabaseHelper.java` – SQLite helper class  
- `StudentCrudActivity.java` – Activity Java file  
- `activity_student_crud.xml` – Activity layout
